Відповіді на контрольні запитання

  1. Мова XHTML- це модифікація мови HTML, яка дозволяє використовувати семантику HTML та строгий синтаксис XML, який гарантує відсутність помилок у вихідному коді та відображання того ж наповнення сторінки на різних пристроях.
  2. Найголовніші правила полягають у наступному:
    Документ обов'язково повинен мати один або більше елементів. В докуметі має бути один і лише один кореневий елемент. Решта елементів ієрархічно піднорядковуються йому. Елементи мають бути iєрархічно коректно підпорядковані між собою. Елементи мають бути записаними в нижньому регістрі. Елементи мають включати початкову та кінцеву частину. Назви атрибутів елементів мають бути в нижньому регістрі. Значення атрибутів мають бути записані у подвіиних лапках.
  3. DOM - це незалежний від платформи і мови програмний інтерфейс, що дозволяє програмам і скриптам отримати доступ до вмісту HTML-, XHTML- і XML-документів, а також його анулювання, структуру і оформлення таких документів. Модель DOM не накладаються обмежень на структуру документа. Будь-який документ відомої структури за допомогою DOM може бути представлений у вигляді дерева вузлів, кожен вузол якого є елементом, атрибут, текстовий, графічний або будь-який інший об'єкт.
  4. HTML5 працює лише в оновлених браузерах.Не є гнучким, як Flash.
  5. Базові атрибути XHTML: id, style, title, class
  6. Кожен блок має область вмісту, в якій знаходиться текст, дочірні елементи, зображення і т.п., і необов'язкові навколишні її padding, border і margin. Розмір кожної області визначається відповідними властивостями і може бути нульовим, або, в разі margin, негативним.Поля, рамка і відступи можуть бути розбиті на верхній, правий, нижній і лівий сегменти, кожен з яких незалежно управляється своїм відповідним властивістю. Фон області вмісту, полів і рамки блоку визначається властивостями фону. Область рамки може бути додатково пофарбована за допомогою властивості border. Відступи елемента завжди прозорі, що дозволяє показувати фон батьківського елемента.
  7. Основні селектори CSS
    1. *- застосувати для усіх елементів сторінки.
    2. #X-застосувати для елемента з id рівним X.
    3. .X-застосувати для елементів, в яких атрибут claas має значення X.
    4. X Y-застосувати для Y, який знаходиться ієрархічномвсередені X.
    5. X-застосувати для всіх елементів, назва тегу яких є Х.
    6. X,Y,Z-застосувати для усіх перерахованих селекторів.
    7. X + Y-застосувати для Y, який знаходитьсяна тому ж рівні ієрархії та розміщенно безпосередньо за Х.
    8. «X:visited», «X:hover» та інші. Застосувати для селектора X зі специфічним станом: посилання, яке вже відвідане, курсор миші над елементом тощо.
    9. «X>Y» застосувати для Y, який знаходиться ієрархічно безпосередньо всередині Х.
    10. «X[Y]» застосувати для X, у якого наявний атрибут Y.
    11. «X[Y=”V”]» застосувати для X, у якого атрибут Y має значення V.
    12. «X[Y$=”V”]» застосувати для X, у якого значення атрибута Y закінчується на V.
